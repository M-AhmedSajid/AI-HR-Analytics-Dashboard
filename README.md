# AI HR Analytics Dashboard

Modern HR Analytics Dashboard built with Next.js App Router, Tailwind CSS, and Recharts.

Local setup

1. Install dependencies:

```bash
npm install
npm install recharts tailwindcss postcss autoprefixer @radix-ui/react-avatar @radix-ui/react-popover
```

2. Initialize Tailwind (if needed):

```bash
npx tailwindcss init -p
```

3. Run dev server:

```bash
npm run dev
```

Notes:
- Components under `src/components` use Tailwind classes and Recharts (charts are client-side only).
- To add shadcn/ui components, follow their installation and generate the components as desired.
